我们抓取到页面之后，还需要对页面进行解析。可以使用字符串处理工具解析页面，也可以使用正则表达式，但是这些方法都会带来很大的开发成本，所以我们需要使用一款专门解析html页面的技术

# 1. jsoup介绍

jsoup是一款Java的HTML解析器，可直接解析某个URL地址、HTML文本内容。它提供了一套非常省力的API，可通过DOM，CSS以及类似于JQuery的操作方法来取出和操作数据

jsoup的主要功能如下：

  1. 从一个URL，文件或字符串中解析HTML
  2. 使用DOM或CSS选择器来查找、取出数据
  3. 可操作HTML元素、属性、文本

导入依赖

<dependency>
    <groupId>org.jsoup</groupId>
    <artifactId>jsoup</artifactId>
    <version>1.14.3</version>
</dependency>

# 2. 解析url

public class JsoupTest {
    public static void main(String[] args) throws Exception {
        // 解析URL地址
        Document dom = Jsoup.parse(new URL("https://www.baidu.com"), 1000);

        // 使用标签选择器,获取title内容
        String title = dom.getElementsByTag("title").first().text();
        System.out.println(title);
    }
}

虽然使用Jsoup可以替代HttpClient直接发起请求解析数据，但是往往不会这样用，因为实际的开发过程中，需要使用到多线程，连接池，代理等等方式，而jsoup对这些的支持并不是很好，所以我们一般把jsoup仅仅作为html解析工具使用

# 3. 解析字符串

public class JsoupTest {
    public static void main(String[] args) throws IOException {
        HttpGet httpGet = new HttpGet("https://www.baidu.com");

        try(CloseableHttpClient httpClient = HttpClients.createDefault();
            CloseableHttpResponse response = httpClient.execute(httpGet)) {
            if (response.getStatusLine().getStatusCode() == 200) {
                // 获取响应体内容
                String html = EntityUtils.toString(response.getEntity(), "utf8");

                // 解析响应体
                Document dom = Jsoup.parse(html);
                
                // 获取title的内容
                String title = dom.getElementsByTag("title").first().text();
                System.out.println(title);
            }
        }
    }
}

# 4. 使用dom遍历文档

# 4.1 元素获取

  1. 根据id查询元素getElementById
  2. 根据标签获取元素getElementsByTag
  3. 根据class获取元素getElementsByClass
  4. 根据属性获取元素getElementByAttribute
public class JsoupTest2 {
    public static void main(String[] args) throws IOException {
        HttpGet httpGet = new HttpGet("https://www.baidu.com");

        try(CloseableHttpClient httpClient = HttpClients.createDefault();
            CloseableHttpResponse response = httpClient.execute(httpGet)) {
            if (response.getStatusLine().getStatusCode() == 200) {
                // 获取响应体内容
                String html = EntityUtils.toString(response.getEntity(), "utf8");

                // 解析响应体
                Document dom = Jsoup.parse(html);

                Element u1 = dom.getElementById("u1");                                  // 通过id获取元素
                System.out.println(u1.text());

                Element title = dom.getElementsByTag("title").first();                  // 通过标签获取元素
                System.out.println(title.text());

                Element mnav = dom.getElementsByClass("mnav").first();                  // 通过class获取
                System.out.println(mnav.text());

                Element maxlength = dom.getElementsByAttribute("maxlength").first();    // 通过属性获取元素
                System.out.println(maxlength.text());
            }
        }
    }
}

# 4.2 数据获取

  1. 从元素中获取id
  2. 从元素中获取className
  3. 从元素中获取属性的值attr
  4. 从元素中获取所有属性的值attributes
  5. 从元素中获取文本内容text
public class JsoupTest {
    public static void main(String[] args) throws Exception {
        HttpGet httpGet = new HttpGet("https://www.baidu.com");

        try(CloseableHttpClient httpClient = HttpClients.createDefault();
            CloseableHttpResponse response = httpClient.execute(httpGet)) {
            if (response.getStatusLine().getStatusCode() == 200) {
                // 获取响应体内容
                String html = EntityUtils.toString(response.getEntity(), "utf8");

                // 解析响应体
                Document dom = Jsoup.parse(html);

                Element element = dom.getElementById("kw");         // 通过id获取元素

                String id = element.id();                           // 获取id
                System.out.println(id);

                Set<String> classNames = element.classNames();      // 获取className
                System.out.println(classNames);

                String name = element.attr("name");     			// 获取指定属性名的属性值
                System.out.println(name);

                Attributes attributes = element.attributes();    	// 获取所有属性
                System.out.println(attributes);

                String text = element.text();                       // 获取文本内容
                System.out.println(text);
            }
        }
    }
}

# 5. 使用选择器遍历文档

# 5.1 Selector概述

tagname：通过标签查找元素，比如：span

#id：通过ID查找元素，比如：#city_bj

.class：通过class名称查找元素，比如.class_a

[attribute]：利用属性查找元素，比如：[abc]

[attr=value]：利用属性值来查找元素，比如：[class=s_name]

public class JsoupTest {
    public static void main(String[] args) throws Exception {
        HttpGet httpGet = new HttpGet("https://www.baidu.com");

        try(CloseableHttpClient httpClient = HttpClients.createDefault();
            CloseableHttpResponse response = httpClient.execute(httpGet)) {
            if (response.getStatusLine().getStatusCode() == 200) {
                // 获取响应体内容
                String html = EntityUtils.toString(response.getEntity(), "utf8");

                // 解析响应体
                Document dom = Jsoup.parse(html);

                Elements spans = dom.select("span");            // 标签选择器
                System.out.println(spans);

                Elements form = dom.select("#form");            // id选择器
                System.out.println(form);

                Elements elements = dom.select(".bg");          // class选择器
                System.out.println(elements);

                Elements names = dom.select("[name]");          // 属性
                System.out.println(names);

                Elements values = dom.select("[value=1]");      // 属性值
                System.out.println(values);
            }
        }
    }
}

# 5.2 选择器组合

表达式 含义 示例
el#id 元素+ID h3#city_bj
el.class 元素+class li.class_a
el[attr] 元素+属性名 span[abc]
任意组合 span[abc].s_name
ancestor child 查找某个元素下子元素 .city_con li
parent > child 查找某个元素下的直接子元素 .city_con > ul > li
parent > * 查找某个元素下所有直接子元素
